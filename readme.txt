# SVG編集ツール
- drawio
    - 説明用の図を作成できる
    - SVGに書き出せる
    - アニメーションの作成ができるが、SVGに変換すると反映されないようだ
    - アニメーションをつけたかったら、直接SVGファイルにアニメーションを設定する
    - 素材からダウンロードしたSVGファイルは、IMGとみなされるため編集不可
    
- vectr
    - ブラウザーで、素材からダウンロードしたSVGファイルでも編集ができる
    - 編集したものは、保存できる
    
- VSCodeの「SVG プラグイン」
    - SVGのコードを直接変更できる
    - 「SVG Viewer」と組み合わせると、コードを見ながらプレビューできる
    - 特にアニメーションを追加する際は役立つ
